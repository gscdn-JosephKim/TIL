# pbcopy, pbpaste
- 커맨드 라인에서 클립보드로 카피하게 해주는 프로그램
- 클립보드의 내용을 커맨드 라인으로 붙여주는 프로그램

# 사용법
```
$ echo 'Hello' | pbcopy
```
```
$ pbpaste
```
